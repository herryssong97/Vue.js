express 설치

cd nodejs 

1. 명령프롬프트에 "npm install express-generator -g" 한 줄을 입력한다.

2. 프로젝트를 만들 경로에서 "express myhome"을 실행한다.

* ex) "express myhome" 이라고 입력하면 myhome 디렉토리가 생성되면서 그 안에 많은 폴더와 파일이 생김.

 create : myhome\
   create : myhome\public\
   create : myhome\public\javascripts\
   create : myhome\public\images\
   create : myhome\public\stylesheets\
   create : myhome\public\stylesheets\style.css
   create : myhome\routes\
   create : myhome\routes\index.js
   create : myhome\routes\users.js
   create : myhome\views\
   create : myhome\views\error.jade
   create : myhome\views\index.jade
   create : myhome\views\layout.jade
   create : myhome\app.js
   create : myhome\package.json
   create : myhome\bin\
   create : myhome\bin\www

   change directory:
     > cd myhome

   install dependencies:
     > npm install

   run the app:
     > SET DEBUG=myhome:* & npm start


3. 프로젝트 디렉토리에서 "npm install" 을 입력해서 package.json의 모듈들을 설치한다.

4. 윈도우는 "set DEBUG=myapp:* & npm start" 를, 
맥&리눅스는 "DEBUG=myapp:* & npm start"를 입력한다.

* 4의 명령을 입력하는 이유는 설치된 애플리케이션의 환경변수에 추가하기 위함이다.


기본적으로 3000 번 포트를 사용한다 

http://127.0.0.1:3000
